Enzyme est la super techno d’Airbnb pour tester à toute bombe des composants React. Elle dispose même d’un plugin Chai pour avoir des assertions plus "métier".

import chaiEnzyme from 'chai-enzyme' import React from 'react'

Le mode “shallow” d’Enzyme est adapté à tout test qui ne nécessite pas le DOM réel mais peut se contenter du DOM virtuel, ce qui inclue même les simulations d’événement.

import { shallow } from 'enzyme' import sinon from 'sinon' import ActionThumbUp from 'material-ui/svg-icons/action/thumb-up' import ContentAdd from 'material-ui/svg-icons/content/add' import Gauge from '../../src/components/gauge' import GoalTrackerWidget from '../../src/components/goal-tracker-widget'
